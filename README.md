# Achaba Analytics â€“ Business Website Concept

Achaba Analytics is a fully responsive, accessibility-focused business website built with modern frontend best practices.  

This project demonstrates structured UI architecture, performance-conscious design, and clean JavaScript state management using only core web technologies.

This is a production-style frontend implementation for a conceptual data analytics consultancy.  

The goal was to build a realistic, scalable business website while emphasizing:

- Clean semantic HTML
- Maintainable CSS architecture
- Accessibility compliance
- Performance optimization
- Structured JavaScript interaction


### Responsive Architecture
- Mobile-first layout
- CSS Grid and Flexbox
- Fluid typography using `clamp()`
- Sticky navigation
- Responsive navigation drawer

### Accessibility Implementation
- Semantic HTML5 structure
- Skip-to-content link
- ARIA attributes and roles
- Keyboard navigation support
- Focus trapping inside modal dialogs
- Escape key handling
- Reduced motion media query support

### Interactive Features
- Mobile menu with state management
- Accessible modal contact form
- Async form submission (Web3Forms API)
- Inline validation with real-time feedback
- Back-to-top scroll detection
- Dynamic year rendering

### Performance Considerations
- Lazy-loaded images
- Minimal layout shifts
- CSS custom properties for maintainability
- Reduced dependency footprint (no frameworks)
- Controlled animation and transition usage

## Tech Stack

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox)
- Vanilla JavaScript (ES6+)
- Web3Forms API

No frameworks or UI libraries.

## What This Project Demonstrates

- Component-based thinking without frameworks
- Clear separation of concerns (HTML / CSS / JS)
- Accessibility-first mindset
- State-driven UI logic
- Form validation architecture
- Production-style code organization
- Performance-aware frontend development
